JV_Products in module

VERSION - 1.83

DESCRIPTION

This module displays products of this category in the module. You can select multiple categories to display. You can display products in a normal module or in a carousel. You can specify a different sort for the output of products. You can display products in sequential or random order. You can display only the recommended, bestsellers, specials, news, popular and with their sort.

Due to its versatility, this module can replace a large number of other modules.

Store the original files when you install the module does not change.
===========================================

FEATURES

	Select a category to display the module in the admin panel
	Choice of several categories to display in the module in the admin panel
	Ability to specify a title for the module. Title can be specified separately for each language that is installed in the store
	Ability to set the picture size for the product
	Ability to set the number of products displayed in the module
	
	Ability to make a pre-selection before sorting:
		recommended products, bestsellers, specials, new products, popular in the selected category
		ability to choose a rating output product (Rev. .. to). For example, display products with a rating from 3 to 5.
		ability to choose count of product output (Rev. .. to). For example, to bring products with the count of 5 to 100.	
		ability to sort products in module:
			collation: name, model, quantity, price, rating, sort order, date added
			by type sort: ascending, descending
	
	Ability to display products in a module in a random order
	Ability to display products in the ordinary type of module or "carousel" (vertical or horizontal carousel, a large number of settings)
	Ability to specify a mapping scheme module (including the home page)
	Multi-Language Support
	Store the original files are not changed
===========================================

COMPATIBILITY

Opencart 1.5.1.x, Opencart 1.5.2.x, Opencart 1.5.3.x, Opencart 1.5.4.1

Ocstore 1.5.1.x, Ocstore 1.5.2.1, Ocstore 1.5.3.1
===========================================

INSTALLATION

1. Copy the contents of the "upload" to the root directory store.
2. In the admin panel set an administrator to edit and view the module.
3. Install the module, and then click "Edit", then set up the module.

The installation and configuration complete.

JV_Products in module - продукты из категорий в модуле

ВЕРСИЯ - 1.88

ОПИСАНИЕ

Этот модуль отображает продукты из указанной категории в модуле. Можно выбрать несколько категорий для отображения. Можно выводить продукты в обычном модуле или в карусели. Можно  задать различную сортировку для выводимых продуктов. Можно выводить продукты в последовательном или случайном порядке. Можно выводить ТОЛЬКО рекомендуемые, хиты продаж, акции, новинки, популярные и при этом их сортировать. 

Благодаря своей универсальности, данный модуль может заменить большое количество других модулей.

Исходные файлы магазина при установке модуля не изменяются.
===========================================

ВОЗМОЖНОСТИ	

	выбор категории для отображения в модуле в админ-панели
	возможность выбора нескольких категорий для отображения в модуле в админ-панели
	возможность указать заголовок для модуля. Заголовок можно указать отдельно для каждого языка, установленного в магазине
	возможность установить размер картинки для продукта
	возможность установить количество выводимых продуктов в модуле
	
	возможность сделать предварительную выборку перед сортировкой
		рекомендуемые продукты, хиты продаж, акции, новинки, популярные в выбранной категории, 
		возможность выбрать рейтинг выводимого продукта (от...до). Например, вывести продукты с рейтингом от 3 до 5.
		возможность выбрать количество выводимого продукта (от...до). Например, вывести продукты с количеством от 5 до 100.
		возможность сортировки продуктов в модуле
			по параметру сортировки: имя, модель, количество, цена, рейтинг, порядок сортировки, дата добавления
			по типу сортировки: возрастание, убывание
	
	возможность отображения продуктов в модуле в случайном порядке
	возможность выводить продукты в обычном типе модуля или в "карусели" (вертикальная или горизонтальная карусель, большое количество настроек)
	возможность указать схему отображения модуля (в том числе на главной странице)
	мультиязычность (английский, русский. Можно добавить другие языки)
	исходные файлы магазина не изменяются
===========================================

УСТАНОВКА

1. Копируем содержимое папки upload в корневую директорию магазина.
2. В админ-панели устанавливаем права администратора на редактирование и просмотр модуля.
3. Устанавливаем модуль, затем нажимаем кнопку "Изменить", затем настраиваем модуль.

На этом установка и настройка закончены.
