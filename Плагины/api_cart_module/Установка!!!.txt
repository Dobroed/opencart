Модуль для работы с apishops.com

Системные требования:

PHP/MySQL

CMS ocStore 1.5.4.1.2



Установка:
1. Устанавливать рекомендуется на свежеустановленную ocStore.
2. Распаковать с заменой содержимое архива в папку с CMS
3. Логин, пароль и id площадки указать в файле catalog/model/checkout/api.php 
4. В админке зайти в "Система-Пользователи-группы пользователей-главный администратор-изменить". Поставить галочки на tool/yml в обоих полях.



Внимание! Если вы собираетесь использовать другие шаблоны дизайна то для нормальной работы необходимо скопировать файл catalog/view/theme/default/template/checkout/cart.tpl в папку catalog/view/theme/ИМЯ_ВАШЕГО_ШАБЛОНА/template/checkout/

Если вдруг после установки что-то не работает проверьте все ли файлы и папки скопировались из архива.
